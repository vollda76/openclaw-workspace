# PowerShell Scripts für Microsoft 365

## 📁 Struktur:

- **01-user-management/** – User erstellen, deaktivieren, Lizenzen
- **02-teams-automation/** – Teams erstellen, Channels, Members
- **03-sharepoint/** – Sites, Berechtigungen, Storage
- **04-security/** – MFA, Audit, Compliance
- **05-monitoring/** – Reports, Alerts, Health Checks
- **templates/** – Vorlagen für CSV, Config

## 🚀 Quick Start:

1. Scripts herunterladen
2. PowerShell Module installieren (siehe INSTALL.md)
3. Mit M365 connecten
4. Script mit -WhatIf testen
5. Produktiv nutzen!
