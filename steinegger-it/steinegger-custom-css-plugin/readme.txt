Steinegger IT Custom CSS Plugin
================================

Installation:
1. Lade den Ordner "steinegger-custom-css" nach /wp-content/plugins/ hoch
2. WordPress Admin → Plugins → "Steinegger IT Custom CSS" aktivieren
3. Fertig! Das CSS wird automatisch geladen.

Ordner-Struktur:
/wp-content/plugins/steinegger-custom-css/steinegger-custom-css.php
